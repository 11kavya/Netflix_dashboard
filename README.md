# 📊 Netflix Data Analysis Dashboard (Power BI)

## 🔍 Project Overview

This project presents an interactive **Netflix Data Analysis Dashboard** built using **Power BI**.
It helps analyze Netflix content such as movies and TV shows, providing insights into genres, ratings, and trends over time.

The dashboard is designed with **interactive slicers and visuals** to make data exploration easy and meaningful.

---

## 🎯 Objectives

* Analyze Netflix content distribution
* Compare content across multiple years
* Identify popular genres and ratings
* Provide interactive filtering for better insights

---

## 📊 Key Features

### 🔹 1. Multi-Year Comparison

* Fiscal Year (FY) slicer supports **multiple selection**
* Allows comparison between years (e.g., 2020 vs 2021)

---

### 🔹 2. Interactive Filters (Slicers)

* Year slicer
* Genre slicer
* Country filter
* Enables dynamic dashboard updates

---

### 🔹 3. Visual Insights

* Distribution of Movies vs TV Shows
* Genre-wise analysis
* Ratings breakdown
* Year-wise content trends

## 🚀 How to Use

1. Download the `.pbix` file
2. Open it using **Power BI Desktop**
3. Use slicers to:

   * Filter by year (multi-select)
   * Explore genres and categories
   * Analyze ratings and trends
